# Sorting Algorithms

This folder contains implementations of basic sorting algorithms in Java as part of my DSA practice journey.

## Implemented Algorithms

### 1. Bubble Sort
**Idea:**  
Compare adjacent elements and swap them if they are in the wrong order. After each pass, the largest element moves to its correct position.

**Time Complexity:**  
- Best Case: O(n)
- Average Case: O(n²)
- Worst Case: O(n²)

**Space Complexity:**  
- O(1)

**Properties:**  
- Stable: Yes  
- In-place: Yes  

---

### 2. Selection Sort
**Idea:**  
Find the minimum element from the unsorted part of the array and place it at the correct position.

**Time Complexity:**  
- Best Case: O(n²)
- Average Case: O(n²)
- Worst Case: O(n²)

**Space Complexity:**  
- O(1)

**Properties:**  
- Stable: No  
- In-place: Yes  

---
